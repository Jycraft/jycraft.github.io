Install

Jycraft's mission includes making all aspects of Minecraft server plugins a lot easier. That includes installation. We still have a long ways to go. Below are instructions for where we're at. In a nutshell:

  • Install Java

  • Download a zip file

  • Run the server

Prerequisites

You need a Spigot-capable version of Java installed. We'll take it from there.

Download

Our releases are distributed from the GitHub release page.

Install

  • unzip the file you downloaded

  • Optional...edit config settings

Running

  • java -r canary_mod.jar

  • In a browser, go to

About Jycraft
Jycraft brings the world of Python to Minecraft server plugins using Jython. Our mission is to provide an environment and ecosytem for programming and sharing using Minecraft.
